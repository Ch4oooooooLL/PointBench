# Project Instructions

- 默认使用中文回复，除非用户明确要求其他语言。
- 当前环境不是虚拟机；完成代码修改后可以运行必要的构建、编译、测试进行验证。
- 不主动运行开发服务器或后端服务；只有用户明确要求时才执行这类命令。
- 前端代码位于 `frontend/`，主要使用 React、TypeScript、Vite。
- 后端代码位于 `backend/`，主要使用 FastAPI、SQLAlchemy、SQLite。
- 修改代码前先阅读相关文件，优先沿用现有组件、服务和样式约定。
- 避免回滚用户已有改动；若工作区有无关变更，保持原样。
- 每次完成一项明确改动后，自动创建本地 Git 提交；推送仍需当前环境具备 GitHub 认证。
